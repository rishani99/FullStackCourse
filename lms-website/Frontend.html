Frontend Development with Full Stack Tools

This page explains a small frontend task, the tools used in a full stack workflow, and how to use them.

Software tools

  • HTML, CSS, JavaScript
  • Code editor: Visual Studio Code, Sublime Text, or similar
  • Browser: Chrome, Edge, Firefox
  • Version control: Git and GitHub
  • Runtime and package manager: Node.js and npm
  • Optional frontend framework: React, Vue, Angular

Small activity / task

Create a simple landing page with a title, description, and a button. Use HTML for structure, CSS for styling, and JavaScript for interaction.

Steps

  1. Open the project folder in your code editor.
  2. Create index.html and add the page structure.
  3. Create styles.css and add styling.
  4. Create script.js to add a click event to the button.
  5. Open the page in the browser and test the button.

How to use the tools

Use the code editor to write and edit the files. Open the browser to view the page and use developer tools to inspect elements and debug. Use Git to save changes and GitHub to share the project.

With Node.js installed, run a local server from the project folder:

npx http-server

Or use the Live Server extension in Visual Studio Code.

Example activity details

Build a frontend page with:

  • A header
  • A paragraph
  • A button that shows an alert when clicked

This is a basic frontend task that fits into a full stack workflow and helps practice the essential tools.
